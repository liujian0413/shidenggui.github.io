另外一种更简洁的语法，花括号语法

作用域

使用var定义的视为局部变量，直接使用的视为全局变量

使用new创建新对象

宿主对象

浏览器预定义的对象(host object), 类似document

window： 浏览器窗口本身
document: 网页内容

节点

<p title='hello'>World</p>
p: 元素节点
title: 属性节点
World: 文本节点

获取元素

  1. getElementById

    返回一个对象，对应着 document 的元素

  2. getElementsByTagName

    document.getElementsByTagName(‘li’) 返回的是一个对象数组
    允许使用通配符 * getElementsByTagName(‘*’)

  3. getElementsByClassName

获得和设置属性

通过元素节点对象调用

getAttribute

setAttribute

object.setAttribute(attribute, value)

非 DOM 解决方案(不推荐)

element.value = ‘the new value’
注: 兼容性较差，只能设置大部分元素的属性

DOM 工作模式

先加载文档的静态内容，在动态刷新，动态刷新不影响文档的静态内容。这正是 DOM 的真正威力: 对页面内容进行刷新却不需要在浏览器里刷新页面。

事件处理函数

this 关键字：指当前的这个对象，在onclick中指拥有onclick属性的对象。

再给某个元素添加事件处理函数后，一旦事件发生，相应的调用代码就会执行。被调用的代码可以返回一个值传递给默认的事件处理函数。
对<a>元素，返回true认为链接被点击了，跳转，返回false什么都不做。
onclik = 'showPic(this); return false;'

childNodes 属性

在一棵节点树上，childNodes 属性可以用来获取任何一个元素的所有子元素，它是包含这个元素全部子元素的数组:
element.childNodes
由 childNodes 属性返回的数组包含所有类型的节点而不仅仅是元素、属性节点、文本节点

nodeType 属性

每一个节点都有 nodeType 属性，通过node的 nodeType 节点确定所需的属性
|节点类型|nodeType值
|—|—
|元素节点|1
|属性节点|2
|文本节点|3

nodeValue

使用 DOM 提供的 nodeValue 属性来获取和设置一个节点的值
node.nodevalue

注: 使用 nodeValue 属性获取 p 元素的值时获取的并不是包含在这个段落的文本，而是 p 元素本身的 value ,其值为 null,包含在 p 元素中的文本是另一种，节点，是 p 元素第一个子节点的 nodeValue 值，即:
p.childNodes[0].nodeValue

firstChild/lastChild

node.firstChild 等价于
node.childNodes[0]
node.lastChild 代表 node 的 childNodes 的最后一个元素

window.onload

window.onload = func;
在页面加载完成后执行 func 函数

window.open

window.open(url, name, features)
在当前浏览器窗口中打开一个新窗口
window.open(winURL, 'popup', 'width=320, height=480');

this

<a href='http://www.ex.com/' onclick='popUp(this.href); return false;'>EX</a>

分离JS

移除 onclick 使用:
element.event = action ...

对象检测

if (!method) return false;
if (!method1 || !method2) return false;

  2. 使用函数

DOM Core
object.getAttribute('name')
HTML DOM
object.name

createElement

var p = document.createElement('p');
若不插入到DOM节点数，称为document fragment文档碎片

appendChild

parent.appendChild(child)

createTextNode

创建一个内容为Hello world的文本节点
var txt = document.createTextNode('Hello world');

在已有元素前插入一个新元素

parentElement.insertBefore(newElement, targetElement)
注:parentElement 等同于 targetElement.parentNode

Ajax

对页面的请求以异步方式发送到服务器。JS动态加载更新内容

XMLHttpRequest 对象

核心，充当浏览器的脚本与服务器之间的中间人的角色，可以代替浏览器发送请求，同时自己处理响应。

var request = new XMLHttpRequest()

open() 方法

指定请求类型: GET / POST / SEND
指定服务器要访问的文件
第三个参数指定请求是否以异步发送和处理

responseText / responseXML 属性

responseText: 保存文本字符串形式的数据
responseXML:用于保存Content-Type头部中指定为’text/xml’的数据，其实是一个DocumentFragment对象

onreadystatechange 事件处理函数

在服务器给XMLHttpRequest对象送回响应时触发执行。

readyState 属性

值 意义
0 未初始化
1 正在加载
2 加载完毕
3 正在交互
4 完成

abbr

<abbr title='Document Object Model'>DOM</abbr> 缩略词

CSS DOM

style 属性

返回一个style对象，保存对应元素节点的样式

当你需要引用一个中间带减号的CSS属性时，DOM使用驼峰命名法
element.style.fontFamily 而不是 element.style.font-family
注: style对象只包含在HTML代码里用style属性声明的样式，基本没有实用价值。
但是可以用来设置样式

className 属性

elem.className等同于elem.getAttribute('class') / elem.setAttribute('class', 'value')

setTimeout

var t1 = setTimeout('function', mstime)
取消执行
clearTimeout(t1)

parseInt / parseFloat 函数

从字符串中返回整数
parseInt(string)
从字符串中返回浮点数
parseFloat(string)

CSS overflow

属性 描述
visible 不裁剪溢出的内容
hidden 隐藏溢出的内容
scroll 隐藏溢出的内容但是显示滚动条
auto 只在溢出时显示滚动条

onmouseover 方法

`object.onmouseover = function () {
//do something
}

jQuery

each

each接受元素在列表中的索引作为参数

$ 专有选择器

选择器 作用
$('tag:even') 匹配元素集中的偶数个元素
$('tag:odd') 匹配元素集中的奇数个元素
$('tag:eq(0))/$('tag:nth(0)') 匹配元素集中的第n个元素
$('tag:gt(n)') 匹配元素集中索引值大于n的所有元素
$('tag:lt(n)') 匹配元素集中索引值小于n的所有元素
$('tag:first') 等价于:eq(0)
$('tag:first') 匹配元素集中的最后一个元素
$('tag:parent') 选择匹配元素集中包含子元素(文本节点也算)的所有元素
$('tag:contains('test')') 选择匹配元素集中包含指定文本的所有元素
`$(‘tag:visible’) 选择匹配元素集中所有可见的元素
$('tag:hidden') 选择匹配元素集中所有隐藏的元素

修改元素

$('p:first').css('font-weight', 'bold');
$('div:hidden').show();
$('div:contains('scared')).hide();

表单元素选择器

选择器 作用
:input 表单所有元素(input,select,textarea,button)
:text 所有文本字段
:password 所有密码字段
:radio 所有单选按钮
:checkbox 所有复选按钮
:submit 所有提交按钮
:image 所有表单图像
:reset 所有重置按钮
:button 所有其他按钮

自定义选择器过滤器

`var singleImageAnchors = $(‘a’).filter(function () {
return ($(‘img’, this).length == 1);
});

生成内容

$('<div id='ex'>Hello</div>').appendTo(document.body);

操作内容

移动元素
$(ul#list1 li').appendTo('ul#list2');
复制元素
$(ul#list li').clone().appendTo('ul#list2');

jQuery and Ajax

$.post(url, params, callback)
通过POST请求取得数据
$.get(url, params, callback)
通过GET请求取得数据
$.getJSON(url, params, callback)
取得JSON对象
$.getScript(url, callback)
取得并执行JavaScript文件

回调方法
接受两个参数，分别是请求对象的响应文本(responseText)和状态(status)
状态是以下几个值之一:

  • success
  • error
  • notmodified

getJSON / getScript 方法的回调函数参数为JS对象

params
为{ key: ‘value’ } 对象
