HTML5 笔记

By shidenggui

發表於 2015-08-29

1、HTML是标记语言

2、常见问题解决

  • 找到问题根源
  • 官方文档
  • 搜索引擎

基础标签

<!DOCTYPE html>
head

  • meta charset 编码格式，默认utf-8
  • title 标题

body 内容

  • h1-h5 标题
  • p 段落(换行)
  • a href 超链接
  • img src 图片

元素、属性、格式化

  • p br 行距不同，p较宽
  • 属性以键/值对的形式出现

常用标签属性

  • a: target规定何处打开链接(“_blank”新页面打开
  • h1: align对其方式
  • body:bgcolor背景颜色background背景图片

通用属性

  • class:规定元素的类名
  • id:规定元素的唯一ID
  • style:规定元素的样式
  • tilte:规定元素的额外信息

HTML格式化

  • b:blod 粗体文本
  • big: 大号字
  • small: 小号字
  • em:着重文章
  • i:italic 斜体
  • strong:加重语气
  • sub:下标字
  • sup:上标字
  • ins:插入字
  • del:删除字

HTML样式

  1. 标签
    • style:样式定义
    • link:资源引用
  2. 属性:
    • rel=”stylesheet”:外部样式表
    • type=”text/css”:引入文档的类型
    • margin-left:边距

样式表插入方法:

  1. 外部样式表:
    <link rel="stylesheet" type="text/css" href="mystyle.css">
  2. 内部样式表
    <style type="text/css"> p {margin-left:20px} </style>
  3. 内联样式表
    <p style="color: red">

HTML 图片、链接、表格

  1. img:
    • alt:无图片时显示文字
  2. a:
    • name:#xxx跳转到xxx
  3. 表格:
    • table:定义表格
      • cellpadding:单元格边距(单元格字体跟单元格边框)
      • cellspacing:单元格间距(单元格之间)
      • bgcolor:背景颜色
      • background:背景图片
    • caption:表头
    • tr:表行
    • td:表单元
    • th:表头

HTML列表

  • ol:有序列表
  • ul:无序列表

  • li:列表项

  • ol:type:

    • disc:实体圆
    • cicle:空心圆
    • square:方块实体
  • ul:
    • type: A,a,I,i标号方式
    • start:定义从哪里开始

自定义列表
dl dt dd

HTML块

  1. 块元素
    • 新行开始:h1 p ul …
  2. 内联元素
    • 不以新行开始:b a img
  3. HTML<div>元素
    • 也称为块元素，组合HTML内容
  4. HTML<span>元素
    • 内联元素，可作为文本容器

HTML布局

  1. 使用<div>布局
  2. 使用table布局

详细参见下面这篇文章

常用表单标签

  • <form> 表单
  • <input> 输入域
  • <textarea> 文本域
  • <lable> 控制标签
  • <fieldset> 定义域
  • <legend> 域的标题
  • <select> 选择列表
  • <optgroup> 选项组
  • <option> 下拉列表中的选项
  • <button> 按钮
    例子: <input type="text">

input 的 type 属性

  • text: 文本
  • password: 密码
  • checkbox: 复选框
  • radio: 单选,需要添加name属性指定组

select:下拉框

textarea

  • cols:列
  • rows:行

button

普通按钮
<input type="button" value="按钮">

提交按钮
<input type="submit" value="提交">

HTML框架

  1. frameset frame(已过时)

    • 不能加在body标签中
    • cols=”x%,x%” 按照x竖向向摆放frame
    • rows=”x%,x%” 按照x横向摆放

  2. iframe

    • frameborder 设置边框
    • 可以嵌套，a标签的_parent父框架打开，_top主页面打开
  • backgroud 背景图片
  • bgcolor 背景颜色
    • red #FF0000
    • green #00FF00
    • blue #0000FF
    • RGB{x,y,z}

在Linux Mint17.2上安装LAMP

By shidenggui

發表於 2015-08-25

更新系统

sudo apt-get update

安装apache2

sudo apt-get install apache2

安装mysql

sudo apt-get install mysql-server libapache2-mod-auth-mysql php5-mysql

激活mysql
sudo mysql_install_db

可选:运行安全脚本(进行安全设置）
sudo /usr/bin/mysql_secure_installation

安装php

sudo apt-get install php5 libapache2-mod-php5

安装php相关模块

查看所有可用模块
sudo apt-cache search php5-

安装指定模块
sudo apt-get install php5-模块名

检验成果

sudo vim /var/www/html/index.php

重启apache2使改动生效
sudo /etc/init.d/apache2 restart 或 sudo service apache2 restart
